# Overview
This script will iterate over a CSV, expecting a very specific format, and
create DNS entries, also in a very specific format, in the specified zone.

## Requirements (tested)
* Python (3.5.3)
* awscli (1.11.109-2.fc25) Fedora
* click (6.7)
* pip (9.0.1)
* setuptools (36.2.4)
* wheel (0.30.0a0)
